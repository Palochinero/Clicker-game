## 🔗 **SERVIDOR LOCAL**

### **🖥️ En tu computadora:**
```bash
# Opción 1: Python
python3 -m http.server 8000

# Opción 2: Node.js
npx serve .

# Opción 3: PHP
php -S localhost:8000
```

### **📱 En tu móvil (misma red WiFi):**
```
http://IP-DE-TU-PC:8000/mobile.html
http://IP-DE-TU-PC:8000/qr-generator.html
```

**🔍 Para encontrar IP de tu PC:**
- **Windows:** `ipconfig`
- **Mac/Linux:** `ifconfig` o `ip addr`

## 📱 **EXPO APP NATIVA**

### **🛠️ Configuración:**

1. **📱 Instala Expo Go:**
   - **iOS:** [App Store](https://apps.apple.com/app/expo-go/id982107779)
   - **Android:** [Google Play](https://play.google.com/store/apps/details?id=host.exp.exponent)

2. **💻 En tu computadora:**
   ```bash
   cd CyberClickerApp
   npx expo start
   ```

3. **📷 Escanea QR** con Expo Go

### **🚀 Para publicar:**
```bash
# Configurar cuenta Expo
expo login

# Build para tiendas
expo build:android  # Google Play
expo build:ios      # App Store
```
